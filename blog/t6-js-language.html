JS Language

In the JavaScript language, describe all the use cases for each of:

parentheses ( )
  1. Execute a function eg. var result = doSomething()
  2. Specifying a function parameters
  3. Grouping to change order of execution eg. 4 * (5+5)
  4. Grouping statements eg. for (i = 0; i < board.cells.length; i++ )
brackets [ ]
  1. Accessing arrays eg. var board = { cell [ ] }
  2. Adding loop conditions eg. board.cells[i].isMine
braces { }
  1. Grouping things inside a function eg. function gameBoard() { }
  2. Creating sections inside a functions
single quotes ' '
  1. Contains strings eg. lib.displayMessage('Your a winner!')
  2. Can be used as quotation marks inside “”
double quotes " "
  1. Again it can contain strings eg. lib.displayMessage(“Your a winner!”)
  2. Can be used as quotation marks inside ‘’
