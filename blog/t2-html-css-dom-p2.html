What is grid based design?

Grid based design is designing structure of your website around a grid of columns and rows. The grids and rows intersect the page which creates sections where you can put your content.

Whats all the hype about responsive webpages?

Responsive websites are all about seeing your website properly on different size screens. Its about setting up your design correctly so the website can be viewed on different devices like a desktop, laptop or cellphone

What is semantic structure?

Basically its about being more specific. Semantic structure is doing away with generic tags like div in favour of more specific ones. Tags like p or h3 describe what is happening inside them. P is a paragraph and h3 is the 3rd biggest or third important heading on the page.
